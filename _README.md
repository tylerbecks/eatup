## Usage

> Some usage instructions

  To start server
  ```
  1. Duplicate db.template.js as db.js
  2. On line 2, change ('database_name', 'username', 'password') to appropriate values.
  3. npm start
  ```

  To compile webpack file automatically 
  ```
  npm run dev
  ```

  To build production webpack file
  ```
  npm run build
  ```
